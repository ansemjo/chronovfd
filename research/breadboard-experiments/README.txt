# Breadboard Experiments

These were some first tests using the HV5812P shift register
and an Adafruit FT232H for SPI communication.

I don't have wiring diagrams anymore, sorry.

| file | function |
| ---- | -------- |
| `helo.py` | simply displays `HELO` in the segments |
| `clock.py` | display the current time with blinking second dots |
